# Flowwy - Premium Focus & Productivity App

**Flowwy** (formerly FocusFlow) is a beautifully designed, premium productivity tool built with Vanilla JS, HTML, and CSS. It combines the Pomodoro technique with curated Lofi beats to help you stay focused and efficient.

Hosted at: [https://flowwy.netlify.app/](https://flowwy.netlify.app/)
Developer Portfolio: [https://khushaankgupta.qzz.io](https://khushaankgupta.qzz.io)

## Features

- **Premium UI**: Glassmorphism design, smooth animations, and a distraction-free "Zen Mode".
- **Productivity Timer**: Customizable Pomodoro, Short Break, and Long Break intervals.
- **Lofi Radio**: Background music powered by YouTube IFrame API (Lofi Girl, Classical, Synthwave).
- **Task Management**: Drag-and-drop support, star priorities, and "Done to Bottom" sorting.
- **PWA Support**: Installable on desktop/mobile with full offline capabilities (App Shell).
- **Gamification**: Built-in "Snake" game to unwind during breaks (with Auto-Pilot bot).
- **Security**: Strict CSP headers, no tracking, locally stored data.

## Deployment

The app is PWA-ready. Simply deploy the files to any static host (Netlify, Vercel, GitHub Pages).

## Technologies Used

- **HTML5 & CSS3**: Custom responsive design with Glassmorphism and CSS animations.
- **JavaScript (Vanilla)**: State management for the timer, tasks, and game logic.
- **YouTube IFrame API**: Seamless integration for music and video backgrounds.

## How to Use

1.  Open `index.html` in your web browser.
2.  Select your timer mode.
3.  Choose your visual theme (Abstract or Video).
4.  Press Play on the music and get to work!
